# WayPlover
*Steno for Wayland*

## Description
A Steno Stroke Interpreter for Wayland.
[Open Steno Project](https://www.openstenoproject.org/plover/)
[Learn Plover]()
## Usage
`wayplover --port /dev/ttyACM0 --dictionary plover.json`

## Features
- [x] Chord History
- [x] Stroke History
- [ ] Dictionary Lookup

## Support
*Only tested with Qmk Keyboard(Planck rev6)*
- [x] Gemini-PR
- [ ] TX-Bolt
